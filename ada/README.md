# Simple Mastermind (ADA version)
Simple player vs computer Mastermind clone, in (very crude) Ada

## How to use
Install GNAT (GCC compiler for ADA):
```bash
$ sudo apt install gnat
```
Compile
```bash
$ gnatmake mastermind.adb
```
Run
```bash
$ ./mastermind
```
