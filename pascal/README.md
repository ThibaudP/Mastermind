# Simple Mastermind (Pascal version)
Simple player vs computer Mastermind clone, in Pascal

## How to use
Install FreePascal
```bash
$ sudo apt install fpc
```
Compile
```bash
$ fpc mastermind.pas -o mastermind
```
Run
```bash
$ ./mastermind
```
